# Tool-X

------------------------------------------------------------------------

### Introduction

*Tool-X 是平时一些小工具存储的仓库


------------------------------------------------------------------------

### 各自的作用

Tool-X 是平时一些小工具存储的仓库:<br>
• **pcd2bin** (把点云的pcd转成bin) <br>
• **sbox2kitti** (把sbox里的激光雷达和图像转成kitti) <br>
• **merge_bag** (a set of python operations used to merging, downsamlping, converting rosbags ) <br>
